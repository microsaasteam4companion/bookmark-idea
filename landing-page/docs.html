1. Download & Install

1Download the ZIP

Click the "Download for Windows" button on the homepage to get AiDocumentAssistant-Windows.zip.

2Extract the folder

Right-click the ZIP file → Extract All. You can place the extracted folder anywhere you like (Desktop, Documents, etc.).

3Run CtrlSense

Open the extracted folder and double-click CtrlSense.exe to launch. No installation wizard is needed — it runs as a standalone portable app.

💡 Tip: If Windows SmartScreen shows a warning, click "More info" → "Run anyway". This appears because the app is not yet code-signed.

2. Adding Documents

There are two ways to add documents into your local search index:

A. Manual Upload

  1. Open the CtrlSense dashboard in your browser (auto-launches at http://localhost:5000).
  2. Click the "Upload" tab in the sidebar.
  3. Drag and drop your PDF or TXT files, or click to browse.
  4. Files are instantly parsed and indexed locally.

B. Watch Folder (Pro Only)

  1. Go to Settings in the dashboard.
  2. Set a Watch Folder path (e.g., C:\Users\You\Documents\AutoIndex).
  3. Any file dropped into this folder is automatically detected, processed, and indexed in the background.
💡 Tip: The Watch Folder feature runs a lightweight background thread that sleeps until it detects new files — it won't slow down your machine.

3. Searching Your Documents

CtrlSense uses semantic AI search — it understands what you mean, not just the exact words you type.

From the Dashboard

  1. Go to the "Search" tab.
  2. Type a natural language query like "invoice from last March" or "contractor agreement penalty clause".
  3. Results appear ranked by relevance with percentage match scores.
  4. Click any result to open the original document, scrolled to the matching section with a golden highlight.

Using the Hotkey (Pro Only)

Press Ctrl + Shift + D from anywhere on your desktop to instantly summon the CtrlSense search overlay — no need to switch windows.

7. Troubleshooting

App won't launch

  • Ensure you extracted the ZIP completely (not running from inside the archive).
  • Right-click CtrlSense.exe → Run as administrator.
  • Check that your system meets the minimum requirements above.

Search returns no results

  • Verify the file was fully indexed (check the Library tab for a file count).
  • Try broader or differently worded queries — semantic search understands synonyms.

SmartScreen warning on launch

  • This is normal for unsigned apps. Click "More info" → "Run anyway".

8. Data & Privacy

CtrlSense is built with a local-first architecture. Your documents, search queries, and AI embeddings never leave your computer. There is no cloud sync, no telemetry, and no tracking. For full details, see our Privacy Policy.
